
UNIDA Admin Role Dashboards Pack

Install:
1. Upload install_admin_role_dashboards.php to project root.
2. Open:
   https://investoraccess.unidatechs.com/install_admin_role_dashboards.php
3. Delete installer after success.

Creates:
- admin/dashboard.php router
- admin/dashboards/super-admin.php
- admin/dashboards/admin.php
- admin/dashboards/verification-admin.php
- admin/dashboards/support-admin.php
- admin/dashboards/finance-admin.php
- admin/dashboards/content-admin.php
- admin/dashboards/partnership-admin.php
- admin/dashboards/analytics-admin.php

Also creates uppercase folder wrappers:
- admin/SUPER_ADMIN/dashboard.php
- admin/VERIFICATION_ADMIN/dashboard.php
- admin/SUPPORT_ADMIN/dashboard.php
- admin/FINANCE_ADMIN/dashboard.php
- admin/CONTENT_ADMIN/dashboard.php
- admin/PARTNERSHIP_ADMIN/dashboard.php
- admin/ANALYTICS_ADMIN/dashboard.php

DB:
- roles
- admin_profiles
- dashboard_registry
