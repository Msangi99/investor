UNIDA Gateway Unny/Rieta Language Switch Pack

Purpose:
- English chatbot assistant = Unny
- Kiswahili chatbot assistant = Rieta
- Global language switch = EN / SW
- Starts with English by default. User can switch to Kiswahili.

Install:
1. Upload install_unny_rieta_language_switch.php to project root.
2. Open:
   https://investoraccess.unidatechs.com/install_unny_rieta_language_switch.php
3. Confirm OK results.
4. Delete:
   install_unny_rieta_language_switch.php

It updates:
- includes/helpers/language.php
- includes/components/language-switch.php
- includes/components/rieta-chatbot-widget.php
- api/rieta-chatbot.php
- assets/js/rieta-chatbot.js
- assets/css/rieta-chatbot.css
- includes/bootstrap.php
- includes/header.php
- includes/footer.php

It creates/updates DB:
- ai_assistants
- ai_settings
- rieta_conversations
- rieta_messages
- rieta_support_requests
